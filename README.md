# Talos Secure AI Chat Agent

A feature-complete, standalone Secure AI Chat application built on [Talos Protocol](https://talosprotocol.com).

## Features
- **End-to-End Encryption**: Double Ratchet (X3DH + Signal)
- **Identity**: DID-based peer identity
- **Audit**: Blockchain-anchored message logs
- **Persistence**: SQLite storage
- **UI**: Modern Next.js interface
